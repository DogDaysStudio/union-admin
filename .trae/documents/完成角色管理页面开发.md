# 角色管理页面开发计划

## 1. 创建角色管理页面文件
- 创建 `src/views/management/role.vue` 文件
- 参考 `permission.vue` 的实现结构

## 2. 实现页面功能
- **查询表单**：根据参考图片实现查询表单字段
- **数据列表**：实现表格显示字段，包括序号、角色编码、角色名称、状态等
- **分页功能**：使用 `usePagination` 实现分页
- **新增/编辑弹窗**：共用一个弹窗，实现表单验证和提交
- **删除功能**：实现角色删除功能，包含确认提示
- **状态切换**：实现角色启用/禁用功能

## 3. 接口调用
- 使用 `iamAuth.iamAuthRoleList` 获取角色列表
- 使用 `iamAuth.iamAuthRoleUpsert` 新增/编辑角色
- 使用 `iamAuth.iamAuthRoleDelete` 删除角色
- 使用 `iamAuth.iamAuthRoleEnable` 启用/禁用角色

## 4. 路由配置
- 在 `src/router/management.ts` 中添加角色管理路由
- 配置路由路径和标题

## 5. 权限设置
- 权限设置部分先占位，不实现具体功能

## 6. 代码规范
- 遵循项目的代码风格规范
- 使用 TypeScript 类型定义
- 按照 Vue 组件规范组织代码

## 7. 测试验证
- 确保页面能正常加载和显示
- 验证新增、编辑、删除功能
- 验证分页和查询功能
