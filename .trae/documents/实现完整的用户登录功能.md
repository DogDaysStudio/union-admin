# 实现完整的用户登录功能

## 1. 实现登录页面组件

### 1.1 创建登录页面组件

* 创建 `src/views/login-view.vue` 文件

* 实现登录表单，包括：

  * 用户名/手机号输入框

  * 密码输入框（带显示/隐藏密码功能）

  * 图形验证码输入框

  * 忘记密码链接

  * 登录按钮

### 1.2 实现表单验证

* 使用 Vue 3 的响应式 API 管理表单数据

* 实现前端验证逻辑：

  * 必填项检查

  * 手机号格式验证

  * 密码格式验证

  * 验证码格式验证

* 显示适当的错误提示

### 1.3 集成登录 API

* 调用 `iamAuthGetImageVerifyCode` 获取图形验证码

* 调用 `iamAuthLogin` 实现登录功能

* 登录成功后获取用户信息并跳转

* 使用 vue-request 来增加loading状态

## 2. 完善路由配置

### 2.1 添加登录路由

* 在 `src/router/index.ts` 中添加登录路由

* 设置登录页面为公共页面，不需要登录即可访问

### 2.2 实现路由守卫

* 启用并完善路由守卫

* 验证用户登录状态

* 限制未登录用户访问受保护页面

* 已登录用户访问登录页时自动跳转

## 3. 完善用户状态管理

### 3.1 优化用户状态管理

* 在 `src/stores/user.ts` 中添加登出方法

* 完善状态管理逻辑

### 3.2 初始化用户信息

* 在应用启动时，从持久化存储中恢复用户状态

* 如果有 token，自动获取用户信息

## 4. 优化 HTTP 服务

### 4.1 完善响应拦截器

* 实现自动登出功能

## 6. 测试和优化

### 6.1 测试登录功能

* 测试正常登录流程

* 测试各种错误情况

* 测试路由守卫功能

### 6.2 优化用户体验

* 优化错误提示

* 完善表单交互

## 实现步骤

1. 首先创建登录页面组件 `src/views/login-view.vue`
2. 然后完善路由配置，添加登录路由和路由守卫
3. 接着优化用户状态管理和 HTTP 服务
4. 最后实现忘记密码功能并进行测试

## 技术要点

* 使用 Vue 3 的 Composition API

* 使用 TypeScript 确保类型安全

* 使用 Pinia 进行状态管理

* 使用 Vue Router 进行路由控制

* 遵循项目的开发规范

* 优先使用 Tailwind CSS 进行样式设计
